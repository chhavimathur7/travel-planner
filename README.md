# ✈️ Travel Planner Website  

A simple and responsive **Travel Planner Website** built using **HTML, CSS, and JavaScript**.  
This website allows users to explore travel destinations, packages, gallery, and even **generate a sample travel plan**.  

🌍 Live Demo: [Click Here](https://chhavimathur7.github.io/travel-planner/)  

---

## 📌 Features  
- 🏖️ **Hero Section** with background image and navigation bar  
- ℹ️ **About Us Section** – Learn more about the travel planner  
- 🌎 **Destinations Section** – Showcase popular travel destinations  
- 🎁 **Packages Section** – Highlight different travel packages  
- 🖼️ **Gallery Section** – Display beautiful travel images  
- 📩 **Contact Us Form** – Simple contact form with validation alert  
- 🗺️ **Generate Plan Feature** – Users can input destination & days to get a sample travel plan  
- 📱 **Responsive Design** – Works on desktops, tablets, and mobiles  
- 🎨 Smooth scrolling & modern UI  

---

## 🛠️ Tech Stack  
- **HTML5**  
- **CSS3** (Flexbox & Grid for layout, custom styling)  
- **JavaScript (Vanilla JS)** – Interactivity, smooth scroll, generate plan feature
